## Featured Projects

### Process Gap Detection Agent
An AI system that analyzes process documents (SOPs and workflows) to detect missing steps, unclear instructions, and logical inconsistencies.

Key Features:
- Uses Small Language Models for structured analysis
- Detects missing or ambiguous process steps
- Generates explainable gap reports
- Designed for workflow optimization and operational efficiency

Technologies:
Python, LLM APIs, NLP, Agent-based logic

---

### Startup Metrics Dashboard
A full-stack analytics dashboard designed to track startup performance metrics and user engagement.

Key Features:
- Real-time analytics visualization
- Product usage metrics tracking
- Interactive dashboards for growth monitoring

Technologies:
React, Firebase, Recharts, Google Analytics API

---

### Risk Score Calculator – Loan Default Prediction
A machine learning project that predicts loan default probability based on financial indicators.

Key Features:
- Data preprocessing and feature engineering
- Risk scoring model
- Analytical visualization of risk factors

Technologies:
Python, Scikit-learn, Pandas, Matplotlib
